📖 Diccionario de Componentes

Catálogo de piezas visuales listas para usar en el proyecto Stocker.

⚛️ Átomos

Atom Button

Botón principal del sistema. Soporta variantes: primary, secondary, outline, ghost y ghost-destructive.

Atom Logo

El identificador visual del proyecto. Soporta modo oscuro y diferentes tamaños.

Atom StatusBadge

Etiqueta visual para estados (Activo/Inactivo) con colores semánticos.

Atom SoundToggle

Botón para habilitar/deshabilitar sonidos. Gestiona el desbloqueo de audio.

🧪 Moléculas

Molecule Modal

Contenedor superpuesto para formularios o detalles. Incluye overlay y cierre.

Molecule StatCard

Tarjeta para mostrar indicadores numéricos en el dashboard (ej. total de productos).

Molecule ToastNotification

Alerta flotante con feedback visual y sonoro.

Molecule SearchInput

Input con icono de búsqueda integrado.

🦠 Organismos

Organism StoreForm

Organismo complejo que gestiona el formulario de tiendas (creación/edición).

Organism Sidebar

Menú lateral de navegación administrativa con enlaces y logo.

Organism AdminHeader

Cabecera del panel que muestra el perfil y el buscador global.
