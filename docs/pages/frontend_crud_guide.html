REGLAS OBLIGATORIAS:
1. Sigue el patrón de Diseño Atómico.
2. El servicio debe usar la instancia central `api`.
3. El formulario debe soportar modos 'create' y 'edit'.
4. La página de listado debe usar `ToastNotification` con sonido.
5. Registra la nueva ruta en `App.tsx` y añádela al `Sidebar.tsx`.
6. Todo el código debe estar excelentemente documentado en español.

Fase 1: Capa de Datos (Servicios)

Paso 1.1 Definir Interfaces y Servicio

Archivo: frontend/src/services/stores.service.ts

import api from './api';

export interface StoreApi {
  id_store: number;
  name: string;
  address: string;
}

export const storesService = {
  getStores: async () => {
    const res = await api.get<StoreApi[]>('/stores/');
    return res.data;
  }
};
