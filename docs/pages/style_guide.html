🎨 Guía de Estilos — Stocker

Referencia visual del sistema de diseño utilizado en la aplicación frontend (React + Tailwind CSS).

0. Configuración del Sistema de Diseño en Código

La base del sistema de diseño está implementada en tres archivos del frontend y versionada en el repositorio:

  • frontend/tailwind.config.js: tokens de color, tipografía y espaciado.
  • frontend/postcss.config.js: pipeline Tailwind + Autoprefixer.
  • frontend/src/index.css: directivas Tailwind y estilos base globales.

Estado actual de la implementación

✅ Correcto: Tailwind está conectado a Vite, PostCSS está activo y los tokens del sistema se consumen desde clases utilitarias.

Fragmentos de referencia

// frontend/postcss.config.js
export default {
    plugins: {
        tailwindcss: {},
        autoprefixer: {},
    },
}
/* frontend/src/index.css */
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
    body {
        @apply bg-surface-base text-ink-base font-sans;
    }
}

2. Tipografía

Fuente principal: Inter (Google Fonts), cargada por Vite y configurada en Tailwind con font-sans.

Jerarquía tipográfica: además de la escala estándar de Tailwind, se definieron tokens semánticos en tailwind.config.js: text-display, text-h1, text-h2, text-h3, text-bodylg, text-body, text-label, text-caption.
Hero (h1) Texto Principal
Sección (h2) Título de Sección
Tarjeta (h3) Título de Tarjeta
Body (lg) Texto de cuerpo grande, para descripciones destacadas.
Body (base) Texto de cuerpo estándar para contenido general.
Small / Labels Etiquetas, textos auxiliares y captions.

6. Efecto Glass (Glassmorphism)

Clase utilitaria .glass definida en index.css. Se usa en la Navbar al hacer scroll y en elementos flotantes del Hero.

Componente con efecto Glass ✨

Definición CSS

.glass {
  background-color: rgb(255 255 255 / 0.7);
  backdrop-filter: blur(16px);
  border: 1px solid rgb(255 255 255 / 0.2);
  box-shadow: 0 10px 15px -3px rgb(0 0 0 / 0.1),
              0 4px 6px -4px rgb(0 0 0 / 0.1);
}

10. Animaciones (Framer Motion)

Se usa framer-motion para transiciones de entrada y micro-animaciones.

PatrónPropiedadesUso
Fade Up initial={{ opacity: 0, y: 20 }}
animate={{ opacity: 1, y: 0 }}		 Títulos, textos, badges del Hero
Scale In initial={{ opacity: 0, scale: 0.95 }}
animate={{ opacity: 1, scale: 1 }}		 Formulario de Login

viewport={{ once: true }}		 Tarjetas de Features
Flotante animate={{ y: [-10, 10, -10] }}
