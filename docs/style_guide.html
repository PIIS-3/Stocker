3. Botones

Bordes redondeados de 12px (rounded-xl). El primario proyecta sombra de marca shadow-brand/30.

Código Tailwind (Primario)

className="bg-brand text-white px-8 py-4 rounded-xl font-semibold
  hover:bg-brand-dark transition-all transform hover:-translate-y-1
  shadow-xl shadow-brand/30"

6. Efecto Glass (Glassmorphism)

Clase utilitaria .glass definida en index.css. Se usa en la Navbar al hacer scroll y en elementos flotantes del Hero.

Componente con efecto Glass ✨

Definición CSS

.glass {
  background-color: rgb(255 255 255 / 0.7);
  backdrop-filter: blur(16px);
  border: 1px solid rgb(255 255 255 / 0.2);
  box-shadow: 0 10px 15px -3px rgb(0 0 0 / 0.1),
              0 4px 6px -4px rgb(0 0 0 / 0.1);
}

10. Animaciones (Framer Motion)

Se usa framer-motion para transiciones de entrada y micro-animaciones.

PatrónPropiedadesUso
Fade Up initial={{ opacity: 0, y: 20 }}
animate={{ opacity: 1, y: 0 }}		 Títulos, textos, badges del Hero
Scale In initial={{ opacity: 0, scale: 0.95 }}
animate={{ opacity: 1, scale: 1 }}		 Formulario de Login
Scroll Reveal whileInView={{ opacity: 1, y: 0 }}
viewport={{ once: true }}		 Tarjetas de Features
Flotante animate={{ y: [-10, 10, -10] }}
repeat: Infinity		 Elementos decorativos del Hero

11. Resumen de Tokens Personalizados

Definidos en tailwind.config.js:

// tailwind.config.js → theme.extend
colors: {
  brand: {
    light:   '#7fc4b7',
    DEFAULT: '#5BA095',
    dark:    '#427c73',
  }
},
fontFamily: {
  sans: ['Inter', 'system-ui', 'sans-serif'],
}

@layer utilities {

    background-color: rgb(255 255 255 / 0.7);
